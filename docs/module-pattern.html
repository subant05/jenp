Parameters:
Name Type Description
config object

This is an object that instructions the function on how to configure the middleware pattern based on behaviors and actions associated with those behaviors. Example: middleware({ behaviors: { send: { submit:function(data,cb){ fetch("/api/submit",{method: post, body: JSON.stringify(data)}) .then(cb,(err)=>throw new Error(err.message)) } } } })
Returns:
  • A middleware manager is returned where one can added add middleware using the use() method. Each middleware must use next() callback as its last argument.
